# AI Automation Blueprint Generator

A local AI assistant built with LangChain and Ollama that generates automation workflows for small businesses.

## Features
- Conversation memory
- Custom system prompt
- Local LLM (no API cost)
- Modular architecture for future expansion

## Tech Stack
- Python
- LangChain
- Ollama (LLaMA3)
- Prompt Engineering

## How to Run

1. Install Ollama
2. Pull model:
   ollama pull llama3

3. Install dependencies:
   pip install -r requirements.txt

4. Run:
   python app.py

## Future Improvements
